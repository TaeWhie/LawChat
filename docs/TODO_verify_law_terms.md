# [나중에 확인] 법령용어 상·하 동일 여부

- **상단** (대분류): 법령용어 — 목록 HTML/XML/JSON, 본문 HTML/XML/JSON
- **하단** (지능형 법령정보지식베이스): 법령용어 XML/JSON, 법령용어-일상용어 연계, 법령용어-조문 등

**확인할 것**: 같은 법령용어 데이터인지, 접근 방식(일반 조회 vs 지능형 연계)만 다른지 → API/오픈데이터 제공처(법제처 등) 문서에서 확인.

**결정**: 중복이면 **아래(지능형)** 쪽을 쓰는 쪽으로. 연계(일상용어·조문) 데이터까지 있어 유동성이 좋음.
**현재**: 사용자가 일단 위·하 모두 체크해 둔 상태. 연동 시 확인 후 하단(지능형) 기준으로 사용.
