## 활용

- **이슈 정규화**: 사용자 질의 → 위 키워드/개념 매핑 (일상용어–법령용어 연계 API 등)
- **판례·해석례 동기화**: `api_data/precedents/` 키워드별 덤프 시 검색어로 사용
- **RAG/상담**: 근로계약·근로시간·임금·휴가·해고·기타 구분으로 단계별 검색 보조
